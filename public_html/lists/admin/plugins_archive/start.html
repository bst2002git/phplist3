Plugins page

The idea here is to build a list of plugins for phpList, similar to the list of Dokuwiki plugins.

To do this, the pluginrepo plugin for Dokuwiki has been cloned with the intention to re-use it for the phpList context.

For now, there are a few incorrect references. Work is in progress.
